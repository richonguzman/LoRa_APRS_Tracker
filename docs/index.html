Firmware Installation

Connect your T-Deck Plus via USB and click the button:

Browser not supported. Use Chrome, Edge, or Chromium. USB access not authorized.
Before installation:
  • Use Google Chrome, Microsoft Edge, or Chromium (Firefox not supported)
  • Connect the T-Deck Plus via USB
  • Close PlatformIO / Arduino IDE if open
  • If port is not detected, hold BOOT button while plugging USB

LVGL UI Features

  • Full touchscreen interface
  • GPS, LoRa, WiFi, Battery dashboard
  • Complete Setup menu
  • APRS / Winlink messages
  • Contact management
  • Physical + virtual keyboard
  • SD card storage
  • APRS symbols
After installation:
  • Configure your callsign via Setup menu or web interface
  • Connect to WiFi AP "LoRa-Tracker-AP" for configuration
  • Documentation: LVGL UI Guide | GitHub CA2RXU
