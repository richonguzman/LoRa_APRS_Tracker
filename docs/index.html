Installation du Firmware

Connectez votre T-Deck Plus en USB et cliquez sur le bouton :

⚠️ Si coché, la configuration existante sera effacée
Navigateur non supporté. Utilisez Chrome ou Edge. Accès USB non autorisé.
Avant l'installation :
  • Utilisez Google Chrome ou Microsoft Edge (Firefox non supporté)
  • Connectez le T-Deck Plus via USB
  • Fermez PlatformIO / Arduino IDE s'ils sont ouverts
  • Si le port n'est pas détecté, maintenez le bouton BOOT en branchant l'USB

Fonctionnalités LVGL UI

  • Interface tactile complète
  • Dashboard GPS, LoRa, WiFi, Batterie
  • Menu Setup complet
  • Messages APRS / Winlink
  • Gestion des contacts
  • Clavier physique + virtuel
  • Stockage SD card
  • Symboles APRS
Après l'installation :
  • Configurez votre callsign via le menu Setup ou l'interface web
  • Connectez-vous au WiFi AP "LoRa APRS Tracker" pour la config
  • Documentation : GitHub CA2RXU
